# Como rodar o projeto
no terminal, digite:
```
cd ["Diretório do projeto + Projeto3\src\Streamlit"]
```

Para instalar as dependências, digite:
```
pip install streamlit
```
Para rodar:
```
streamlit run app.py
```

Abrirá uma página no navegador com o projeto rodando.
